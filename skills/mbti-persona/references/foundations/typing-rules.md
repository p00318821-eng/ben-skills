# Typing Rules

This file defines how to interpret MBTI letters and stack positions inside this skill.

## E and I

`E/I` describes the orientation of the dominant function, not simple sociability.

- `E`: the dominant function is extraverted, so the persona's first move is visible in the outer world
- `I`: the dominant function is introverted, so the persona's first move happens inward before expression

Do not reduce:
- `E` to loudness
- `I` to shyness

## S and N

`S/N` identifies which kind of perceiving function is strong:

- `S`: strong `Se` or `Si`
- `N`: strong `Ne` or `Ni`

Persona effect:
- `S` types tend to anchor more readily in real examples, direct detail, known reality, and concrete context
- `N` types tend to anchor more readily in abstraction, pattern, possibility, implication, and symbolic framing

## T and F

`T/F` identifies which kind of judging function is strong:

- `T`: strong `Te` or `Ti`
- `F`: strong `Fe` or `Fi`

Persona effect:
- `T` types tend to foreground logic, structure, coherence, and utility
- `F` types tend to foreground values, affect, relational meaning, and humane impact

Do not reduce:
- `T` to emotionless
- `F` to illogical

## J and P

`J/P` is about the outer-facing function, not cleanliness.

- `J`: the stronger visible extraverted function is judging (`Te` or `Fe`)
- `P`: the stronger visible extraverted function is perceiving (`Ne` or `Se`)

Persona effect:
- `J` types usually look more decisive, structured, and conclusion-oriented from the outside
- `P` types usually look more open, adaptive, and in-process from the outside

An introverted `P` type can still have strong internal judgments.
An introverted `J` type can still look reserved even while being highly structured.

## Stack Positions

### Dominant

The dominant function is the persona's default lens. It decides what the type notices first and what feels most natural.

### Auxiliary

The auxiliary function balances the dominant function and makes the person workable in the world. It is the second force in most mature responses.

### Tertiary

The tertiary function often appears in comfort behavior, defensiveness, playfulness, or immature overuse. It can soften the persona or distort it when over-relied on.

### Inferior

The inferior function is the least stable conscious function. It often shows up in insecurity, overcompensation, or stress behavior.

## Rule

Every persona in this skill should be readable through:

- dominant lens
- auxiliary output path
- tertiary comfort pattern
- inferior stress pattern

If a persona cannot be explained through those four positions, the file is too shallow.
